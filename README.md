Design of a two-stage machine learning pipeline to first classify whether a song would become a hit and subsequently predict its popularity score using Spotify audio features. Performing a feature engineering and preprocessing based on findings from the literature, including feature selection, popularity thresholding, correlation analysis, multicollinearity assessment, and UMAP visualisation for exploratory analysis. Comparison of multiple classification and regression models, including Logistic Regression, MLP, XGBoost, ResNet, and Neural Network + Random Forest hybrid models, comparing predictive performance across both stages. Use of XAI techniques (SHAP, Partial Dependence Plots (PDP), and Individual Conditional Expectation (ICE)) to interpret model predictions and identify the audio characteristics most influential in determining song popularity.
